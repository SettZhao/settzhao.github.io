# 🍳 菜谱系统 - 快速上手指南

## 🎉 现在添加菜谱超简单！

你只需要做两件事：
1. **编辑一个JS文件**（添加菜谱数据）
2. **放一张图片**（可选）

就这么简单！页面会自动生成。

---

## 📝 第一步：添加菜谱数据

打开文件：`recipes/recipes-data.js`

在 `recipesData` 数组中，复制下面的模板，填写你的菜谱信息：

```javascript
{
    id: "caipin-pinyin",              // 菜品拼音ID（用于URL，不要重复）
    name: "你的菜名",                  // 菜品名称
    emoji: "🍖",                      // 代表这道菜的emoji
    subtitle: "一句话介绍",            // 副标题，简短描述
    time: "30分钟",                    // 烹饪时间
    servings: "2-3人份",              // 份量
    difficulty: "★★☆",               // 难度（1-5颗星）
    description: "详细描述这道菜的特点...",  // 详细介绍
    tags: ["标签1", "标签2", "标签3"],  // 菜品标签
    image: "images/你的菜名.jpg",      // 图片路径（可选）
    
    // 用料清单
    ingredients: [
        { name: "食材1", amount: "用量" },
        { name: "食材2", amount: "用量" }
    ],
    
    // 烹饪步骤
    steps: [
        "第一步的详细描述...",
        "第二步的详细描述...",
        "第三步的详细描述..."
    ],
    
    // 小贴士
    tips: [
        "第一条烹饪技巧...",
        "第二条注意事项...",
        "第三条经验分享..."
    ]
},
```

### 💡 填写要点：

**ID命名规则：**
- 用拼音或英文
- 全小写
- 不要有空格和特殊字符
- 例如：`hongshaorou`, `fanqiechaoDan`, `gongbaojiding`

**难度星级：**
- ★☆☆ - 新手友好
- ★★☆ - 有一定难度
- ★★★ - 需要技巧
- ★★★★ - 比较复杂
- ★★★★★ - 大厨级别

**常用标签：**
- 菜系：川菜、粤菜、湘菜、东北菜、家常菜
- 特点：快手菜、下饭神器、待客首选、硬菜
- 适合：新手友好、进阶菜、周末大餐
- 口味：麻辣、酸甜、清淡、香辣、鲜香

---

## 📸 第二步：添加菜品图片（可选）

1. 把菜品照片重命名为菜名，例如：`红烧肉.jpg`
2. 放到文件夹：`recipes/images/`
3. 在数据中填写：`image: "images/红烧肉.jpg"`

**图片要求：**
- 格式：.jpg, .jpeg, .png, .webp 都可以
- 尺寸：建议至少 800x600 像素
- 如果暂时没有图片，系统会显示emoji占位

## ❓ 常见问题

**Q: 添加菜谱后页面没变化？**
A: 检查JS语法是否正确，特别是逗号、引号、大括号。可以按F12打开控制台看是否有错误。

**Q: 图片显示不出来？**
A: 检查图片路径是否正确，图片是否已放入images文件夹，文件名是否匹配。

**Q: 想删除某个菜谱怎么办？**
A: 在 recipes-data.js 中找到对应的菜谱数据，删除整个 {...} 对象即可。

**Q: 能改变页面颜色吗？**
A: 可以！编辑 index.html 和 detail.html 中的 CSS 部分。

**Q: 可以导出菜谱吗？**
A: recipes-data.js 本身就是标准的JSON格式，可以随时备份或迁移。

## 🚀 高级技巧

### 1. 批量添加菜谱
直接在 recipes-data.js 中复制多个菜谱对象，一次性添加。

### 2. 使用在线图片
如果不想本地存储图片，可以用图床链接：
```javascript
image: "https://你的图床地址/菜名.jpg"
```

### 3. 自定义标签颜色
在CSS中添加特定标签的样式（需要编辑HTML文件）。
